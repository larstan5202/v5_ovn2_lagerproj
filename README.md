# Lagerhanteringssystem
# Översikt

Lägga till produkter
Minska antalet av en specifik produkt
Testas enlig BDD-principen(Given / When / Then)

Produkten består av
stock.py
tests/test_stock.py

## Projektstruktur

│
├── stock.py

StockIten reprensterar en produkt i lagret.

name - produktens namn

Stock en lista av produkter
